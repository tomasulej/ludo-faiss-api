Labels

Form labels can be created by wrapping <label> into element with class .form-label. Form label can be also placed beside inputs or stack as shown in form layouts.

Example:
Code:
<!-- input -->
<div class="form-control form-control--input">
  <div class="form-label">
    <label for="...">Form label:</label>
  </div>
  <input type="text" class="input" id="...">
</div>

<!-- textarea -->
<div class="form-control form-control--textarea">
  <div class="form-label">
    <label for="...">Form label:</label>
  </div>
  <textarea type="text" class="input" id="..." rows="3">
</div>

<!-- checkbox -->
<div class="form-control form-control--checkbox">
  <div class="form-label">
    <label for="...">Form label:</label>
  </div>
  <div class="checkbox">...</div>
</div>

<!-- radiogroup -->
<div class="form-control form-control--group">
  <div class="radiocheck-group">
    <div class="form-label">
      <label>Form label:</label>
    </div>
    <div class="form-control form-control--radio">
      <div class="radio">...</div>
    </div>
    ...
  </div>
</div>

<!-- select -->
<div class="form-control form-control--select">
  <div class="form-label">
    <label>Form label:</label>
  </div>
  <select class="js-select">...</select>
</div>

Form label inline

Inline form labels can be created with form control modifier .form-control--inline.

Note

Long labels can have max-width set as inline style attribute to wrap text.

Warning

Inline labels are currently supported on .radiocheck-group element.

Example:
Code:
<div class="form-control form-control--input">
  <div class="form-label">
    <label for="...">Form label:</label>
  </div>
  <input type="text" class="input" id="...">
</div>

Form label states

Form labels can have two state: default and error state. Error state can be set by using class .is-error on .form-label element.

Example:
Code:
<!-- default -->
...
  <div class="form-label">
    <label for="...">Form label:</label>
  </div>
...

<!-- error -->
...
  <div class="form-label error">
    <label for="...">Form label:</label>
  </div>
...

Form label sizes

There are three default sizes of form labels: medium (default), small and large. These can be set by using classes .form-label--[large/small].

Example:
Code:
<!-- large -->
...
  <div class="form-label form-label--large">
    <label for="...">Form label large:</label>
  </div>
...

<!-- medium -->
...
  <div class="form-label">
    <label for="...">Form label medium:</label>
  </div>
...

<!-- small -->
...
  <div class="form-label form-label--small">
    <label for="...">Form label small:</label>
  </div>
...

Form text

In case you need to show additional help text for form components you can use form text component. Help text will be then shown under form component. To create form text component you can use <p> tag with class .form-text as shown in example below.

Example:

Lorem ipsum Elit anim tempor aliquip tempor mollit.

Lorem ipsum Elit anim tempor aliquip tempor mollit.

Lorem ipsum Sit eu velit.

Lorem ipsum Anim nisi et in dolor.

Lorem ipsum Sed est culpa nulla sit laboris sed aliqua aute nostrud dolore consectetur dolor sed dolore dolore et dolor tempor laboris.

Code:
<!-- input -->
<div class="form-control form-control--input">
  <input type="text" class="input" id="...">
  <p class="form-text">...</p>
</div>

<!-- textarea -->
<div class="form-control form-control--textarea">
  <textarea type="text" class="input" id="..." rows="3">
  <p class="form-text">...</p>
</div>

<!-- checkbox -->
<div class="form-control form-control--checkbox">
  <div class="checkbox">...</div>
  <p class="form-text">...</p>
</div>

<!-- radiogroup -->
<div class="form-control form-control--group">
  <div class="radiocheck-group">
    <div class="form-control form-control--radio">
      <div class="radio">...</div>
    </div>
    ...
    <p class="form-text">...</p>
  </div>
</div>

<!-- select -->
<div class="form-control form-control--select">
  <select class="js-select">...</select>
  <p class="form-text">...</p>
</div>
...

Form tooltips

Validation tooltips for providing real-time input feedback.

Example:
Validation message
Success validation message
Warning validation message
Success large inline validation message
Error inline validation message
Rich validation message, which is multiline, and has some link and maybe another
Code:
<div class="form-control form-control--input">
  <div class="form-label">
    <label for="input-tooltip">Form label:</label>
  </div>
  <div class="form-tooltip-wrapper">
    <input class="input" type="text" placeholder="Input" id="input-tooltip">
    <div class="form-tooltip active">Validation message</div>
  </div>
</div>

<div class="form-control form-control--input">
  <div class="form-label is-valid">
    <label for="input-success-tooltip">Form label:</label>
  </div>
  <div class="form-tooltip-wrapper">
    <input class="input is-valid" type="text" placeholder="Input" id="input-success-tooltip">
    <div class="form-tooltip form-tooltip--success form-tooltip--medium-large active">Success validation message</div>
  </div>
</div>

<div class="form-control form-control--input">
  <div class="form-label">
    <label for="input-warning-tooltip">Form label:</label>
  </div>
  <div class="form-tooltip-wrapper">
    <input class="input" type="text" placeholder="Input" id="input-warning-tooltip">
    <div class="form-tooltip form-tooltip--warning form-tooltip--medium-large active">Warning validation message</div>
  </div>
</div>

<div class="form-control form-control--checkbox">
