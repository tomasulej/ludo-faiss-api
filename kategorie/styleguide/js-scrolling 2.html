Scrolling

Offset

All modules which uses scroll for interaction, e.g. smooth scrolling, scroll spying, element sticking or setting scroll position based on hash in url uses [data-scroll-offset] attribute to specify offset from window top.

Default value for whole page, which can be overridden localy with [data-scroll-offset], can be set with window.myApp.scrollOffset.

[data-scroll-offset] can be specified as number (of pixels) or responsive map with key as breakpoint (in pixels) and value as offset. Example is shown below.

Code:
<a href="#" data-scroll data-scroll-offset="85" />
<!-- or responsive -->
<a href="#" data-scroll data-scroll-offset="{"0": "85", "768": "95"}" />

<script>
  window.myApp = window.myApp = {};
  window.myApp.scrollOffset = 85;
  // or responsive
  window.myApp.scrollOffset = {0: 85, 768: 95};
</script>
